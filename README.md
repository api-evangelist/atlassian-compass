# Atlassian Compass (atlassian-compass)
Atlassian Compass is a developer experience platform that helps engineering teams understand, manage, and improve the health of their software components and services. It provides a centralized catalog of software components with scorecards, metrics, and dependency tracking to improve developer productivity and software quality.
